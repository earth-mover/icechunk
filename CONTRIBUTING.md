# Contributing

The Icechunk project welcomes contributions from anyone. Including:

- Bug reports
- Feature requests
- Documentation PRs
- Bug Fixes
- New Features

For more details on how to contribute please see our contributing guide: <https://icechunk.io/en/latest/reference/contributing/> or the local copy in `icechunk-python/docs/docs/reference/contributing.md`

If you have questions or want to discuss a contribution, join us in the [Earthmover Community Slack](https://join.slack.com/t/earthmover-community/shared_invite/zt-2cwje92ir-xU3CfdG8BI~4CJOJy~sceQ).
