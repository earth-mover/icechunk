# Agent Instructions

If you are an AI agent preparing a contribution to this repository, read and follow the
[AI Usage Policy](icechunk-python/docs/docs/ai-policy.md) before submitting any changes.

Key points:

- A human must review, understand, and take responsibility for every line of code you produce.
- All PR descriptions and review comments must be written by the human, not generated by you.
- Disclose AI tool usage in the PR description.
- Do not submit PRs autonomously — a human must be directly involved in the submission.
- For large changes, the human should open an issue first to discuss scope with maintainers.
